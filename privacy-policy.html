Privacy Policy

Extension Name: HVS Mass Unsub

Effective Date: January 2026

This extension does NOT collect, store, transmit, or share any personal data.

1. Overview

HVS Mass Unsub is a Chrome Extension designed to help users manage and unsubscribe from YouTube channels efficiently. This extension operates entirely within the user's browser and does not transmit any data to external servers.

2. Information We Collect

We do not collect any of the following:

  • No personal information
  • No Google account data
  • No browsing history
  • No cookies
  • No analytics data
  • No tracking information
  • No IP addresses

3. How the Extension Works

The extension:

  • Accesses the YouTube subscriptions page locally within your browser
  • Injects a user interface for managing subscriptions
  • Automates unsubscribe actions based on user selection
  • Does not store or transmit subscription data

All processing happens locally in your browser session.

4. Permissions Usage

The extension may request the following permissions:

  • Tabs Permission: To detect or open the YouTube subscriptions page.
  • Scripting Permission: To inject the management interface into YouTube.
  • Host Permission (youtube.com): Required to interact with the subscription page.

These permissions are used strictly for core functionality and nothing else.

5. Data Storage

The extension does not store:

  • User credentials
  • Subscription lists
  • Account identifiers
  • Activity logs

No data is saved locally or remotely.

6. Third-Party Services

This extension does not use:

  • External APIs
  • Analytics providers
  • Advertising services
  • Tracking scripts

7. Security

Because the extension does not collect or transmit data, there is no risk of data exposure from external servers. All actions occur within your authenticated YouTube session.

8. User Responsibility

Users are responsible for their own YouTube account actions. The extension simply automates unsubscribe clicks within the YouTube interface.

9. Changes to This Policy

If functionality changes in the future, this Privacy Policy will be updated accordingly.

10. Contact

For questions regarding this Privacy Policy, please contact the developer via the Chrome Web Store listing.
